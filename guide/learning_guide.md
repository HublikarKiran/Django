# Learning App Guide

The learning app manages subjects, study materials, assignments, and assignment submissions.

Important URLs:

- Subjects: `/learning/`
- Create subject: `/learning/subjects/create/`
- Materials: `/learning/materials/`
- Upload material: `/learning/materials/create/`
- Assignments: `/learning/assignments/`
- Create assignment: `/learning/assignments/create/`
- Submit assignment: `/learning/assignments/<id>/submit/`

Admins create subjects/materials/assignments. Students view learning content and submit assignments.
